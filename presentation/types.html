Types of Authentication

Session-Based Authentication

Сервер поддерживает сессию для каждого аутентифицированного пользователя.

Token-Based Authentication (e.g., JWT)

После успешной аутентификации сервер выдает токен (JWT), содержащий данные о пользователе.

Federated Authentication (e.g., OAuth)

Процесс аутентификации делегируется внешнему провайдеру (например, Google).
